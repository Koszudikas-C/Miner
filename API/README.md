# API

Este diretório contém a implementação dos serviços de API, incluindo a `ApiRemoteWorkClientBlockChain`, responsável pela comunicação remota e orquestração de operações de blockchain com clientes.

## Estrutura
- **ApiRemoteWorkClientBlockChain/**: Implementação dos endpoints, serviços, gerenciadores de conexão e lógica de negócio.
- **API.sln**: Solução Visual Studio para build e organização do projeto.

## Principais componentes
- Controllers REST
- Serviços de gerenciamento de clientes e conexões
- Integração com bibliotecas compartilhadas

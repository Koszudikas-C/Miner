# Test

Contém projetos de teste automatizado para cada módulo principal do sistema. Inclui testes unitários, de integração e mocks de serviços.

## Estrutura
- **TestApiRemoteWorkClientBlockChain/**: Testes para a API de blockchain
- **TestGlobalEventBus/**: Testes para o barramento de eventos
- **TestWorkClientBlockChain/**: Testes para o cliente/trabalhador
- **Test.sln**: Solução Visual Studio para build dos testes

## Boas práticas
- Separe testes unitários e de integração
- Use mocks para dependências externas
- Garanta cobertura mínima para funcionalidades críticas

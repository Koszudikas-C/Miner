# LibClass

Contém bibliotecas utilitárias e de infraestrutura usadas por múltiplos módulos do projeto. Inclui funcionalidades de comunicação, manipulação de arquivos, criptografia, sockets, SSL, etc.

## Submódulos
- **LibCertificate/**: Gerenciamento de certificados
- **LibCommunicationStatus/**: Status e eventos de comunicação
- **LibHandler/**: Manipuladores genéricos
- **LibJson/**: Serialização e deserialização JSON
- **LibLayout/**: Layouts e templates
- **LibMiddleware/**: Middlewares reutilizáveis
- **LibReceive/**: Recebimento de dados
- **LibRemoteAndClient/**: Entidades e contratos para comunicação remota
- **LibSearchFile/**: Busca e manipulação de arquivos
- **LibSend/**: Envio de dados
- **LibSocket/**: Abstração de sockets
- **LibSocketAndSslStream/**: Integração socket + SSL
- **LibSocks5/**: Proxy SOCKS5
- **LibSsl/**: Utilitários SSL
- **LibTimeTask/**: Agendamento de tarefas
